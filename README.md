# SimpleTodoApp-MLH

Simple Android to-do list app built with Kotlin and Jetpack Compose.

## Features
- Add new tasks from the main screen
- View a list of existing tasks
- Clear tasks when they are completed

## Tech stack
- Kotlin
- Jetpack Compose
- Android Studio

## How to run
1. Clone this repository.
2. Open it in Android Studio.
3. Build and run the `app` module on an emulator or Android device.
